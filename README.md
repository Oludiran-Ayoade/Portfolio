# Portfolio

Personal portfolio website for **Oludiran-Ayoade Olutimileyin Emmanuel** — Full Stack Developer.

Built with Next.js 14, TypeScript, TailwindCSS, and Lucide React icons.

## Sections

- Hero
- About
- Work Experience
- Skills
- Projects
- Education
- Certifications
- Contact

## Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Configured for Netlify via `netlify.toml`.
